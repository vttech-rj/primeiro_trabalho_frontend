Desvendando a semântica do HTML.

Neste artigo, iremos encontrar algumas tag semânticas.

  1. HEADER
  2. NAV
  3. MAIN
  4. ASIDE
  5. ARTICLE
  6. FOOTER

- HEADER

A tag header representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

A tag header normalmente contém elementos (h1 - h6), logotipo ou ícone e informações de autoria

- NAV

A tag nav define um conjunto de links de navegação.

Observe que nem todos os links de um documento devem estar dentro de uma tag nav. A tag nav destina-se apenas aos principais blocos de links de navegação.

Os navegadores, como leitores de tela para usuários com deficiência, podem usar esse elemento para determinar se devem omitir a renderização inicial desse conteúdo.

- MAIN

A tag mainespecifica o conteúdo principal de um documento.

O conteúdo dentro da tag main deve ser exclusivo do documento. Ele não deve conter nenhum conteúdo repetido em documentos como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa.

Não deve haver mais de uma tag main elemento em um documento.

- ASIDE

A tag aside define algum conteúdo além do conteúdo em que é colocado.

O conteúdo de lado deve estar indiretamente relacionado ao conteúdo ao redor.

- ARTICLE

A tag article especifica conteúdo independente e autocontido.

A tag article deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

- FOOTER

A tag footer define um rodapé para um documento ou seção.
